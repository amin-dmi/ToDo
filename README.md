# ToDo Flutter App

## توضیحات
پروژه‌ی ToDo یک اپلیکیشن ساده بر اساس فریم‌ورک فلاتر است که به شما امکان مدیریت وظایف روزانه‌تان را می‌دهد.

## ویژگی‌ها
- افزودن و حذف وظایف
- نشانه‌گذاری وظایف به عنوان تکمیل شده
- دسته‌بندی وظایف به صورت گروه‌بندی شده
